OpenSSL commands to generate SSL certificate

1.openssl genrsa -out privatekey.pem 1024 => generate private key

2.openssl req -new -key privatekey.pem -out certrequest.csr -config C:\OpenSSL-Win64\bin\openssl.cfg => generate certificate

3.openssl x509 -req -in certrequest.csr -signkey privatekey.pem -out certificate.pem => generate self-signed certificate

these 2 files privatekey.pem and certificate.pem are generated in bin folder 
to  use these 2 files keep it i keys folder of your application.
